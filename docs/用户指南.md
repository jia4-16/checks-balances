## 产品介绍

个人记账工具是一款轻量级、易用的个人财务管理应用，帮助您：

- ✅ 记录日常支出
- ✅ 设置和跟踪预算
- ✅ 分析消费趋势
- ✅ 优化消费结构

### 核心特性

- 🎨 简洁美观的界面
- 📊 直观的数据可视化
- 💾 本地数据存储
- 📱 响应式设计
- 🔒 隐私优先

## 功能详解

### 1. 支出记录管理

#### 1.1 添加支出

**操作步骤**：
1. 点击右上角的 **"快速记账"** 按钮
2. 在弹出的表单中填写：

| 字段 | 说明 | 必填 |
|-----|------|------|
| 金额 | 支出金额，精确到分 | ✅ |
| 类别 | 选择支出类别 | ✅ |
| 日期 | 交易日期，默认今天 | ✅ |
| 备注 | 可选的详细说明 | ❌ |

3. 点击 **"保存"** 完成

**预设类别**：
- 🍔 餐饮 - 早餐、午餐、晚餐、零食等
- 🚗 交通 - 公交、地铁、打车、加油等
- 🏠 住宿 - 房租、水电费、物业费等
- 🎮 娱乐 - 电影、游戏、旅游等
- 🛍️ 购物 - 衣服、日用品、电子产品等
- 💊 医疗 - 药品、看病、体检等
- 📚 教育 - 书籍、课程、培训等
- 📱 通讯 - 话费、网费、流量等
- 🎁 人情往来 - 礼物、红包、请客等
- 📦 其他 - 不属于上述类别的支出

#### 1.2 编辑支出

**操作步骤**：
1. 在支出列表中找到要编辑的记录
2. 点击记录右侧的 ✏️ **编辑** 按钮
3. 修改需要更新的字段
4. 点击 **"保存"**

#### 1.3 删除支出

**操作步骤**：
1. 在支出列表中找到要删除的记录
2. 点击记录右侧的 🗑️ **删除** 按钮
3. 在确认对话框中点击 **"删除"**

⚠️ **重要**：删除操作不可恢复，请谨慎操作！

### 2. 预算管理

#### 2.1 设置预算

**操作步骤**：
1. 点击 **"设置预算"** 按钮
2. 输入月度预算总额（单位：元）
3. 点击 **"保存"**

💡 **设置建议**：
- 根据月收入设置，建议不超过月收入的 80%
- 预留 20% 作为应急资金
- 可以根据实际情况随时调整

#### 2.2 预算指标说明

| 指标 | 计算方式 | 说明 |
|-----|---------|------|
| 预算总额 | 用户设置 | 您设定的月度预算 |
| 已支出 | 本月支出总和 | 本月已花费的金额 |
| 剩余预算 | 预算总额 - 已支出 | 还可以花多少钱 |
| 预算使用率 | (已支出 ÷ 预算总额) × 100% | 预算使用百分比 |
| 每日可支配 | 剩余预算 ÷ 当月剩余天数 | 平均每天可以花多少 |

#### 2.3 预算状态提示

系统会根据预算使用率自动显示状态提示：

| 状态 | 使用率 | 提示 | 颜色 |
|-----|-------|------|------|
| 预算充足 | < 80% | ✅ 预算充足，还剩 X 天可支配 | 绿色 |
| 预算紧张 | 80% - 99% | ⚠️ 预算紧张！已使用 X%，请注意控制支出 | 黄色 |
| 预算超支 | ≥ 100% | 🚨 预算超支！已超支 ¥X.XX | 红色 |

### 3. 统计分析

#### 3.1 分类占比饼图

- **用途**：直观展示各支出类别的金额占比
- **数据范围**：本月支出
- **交互**：悬停查看具体金额
- **颜色**：使用不同颜色区分各类别

#### 3.2 每日支出趋势折线图

- **用途**：展示本月每日支出的变化趋势
- **数据范围**：本月有支出的日期
- **交互**：悬停查看当日具体支出
- **特点**：平滑曲线，便于观察趋势

### 4. 数据筛选

#### 4.1 按类别筛选

1. 点击支出记录上方的类别下拉菜单
2. 选择要查看的类别
3. 列表自动更新显示该类别的记录

选择"所有类别"可取消筛选。

#### 4.2 按时间筛选

1. 点击时间下拉菜单
2. 选择时间范围：
   - **全部时间**：显示所有记录
   - **今天**：只显示今天的记录
   - **本周**：显示最近 7 天的记录
   - **本月**：显示本月的记录（默认）

### 5. 数据管理

#### 5.1 数据存储

所有数据存储在浏览器的 localStorage 中：

| 数据项 | Key | 说明 |
|-------|-----|------|
| 支出记录 | expenses | JSON 数组 |
| 预算设置 | budget | JSON 对象 |
| 类别配置 | categories | JSON 数组 |

#### 5.2 查看原始数据

如需查看或备份原始数据：

1. 按 `F12` 打开开发者工具
2. 切换到 **Console** 标签
3. 输入以下命令：

```javascript
// 查看支出记录
localStorage.getItem('expenses')

// 查看预算设置
localStorage.getItem('budget')

// 备份所有数据
JSON.stringify({
  expenses: JSON.parse(localStorage.getItem('expenses') || '[]'),
  budget: JSON.parse(localStorage.getItem('budget') || '{"total":0}'),
  categories: JSON.parse(localStorage.getItem('categories') || '[]')
})
```

#### 5.3 清除所有数据

如需重置应用：

```javascript
// 在 Console 中执行
localStorage.clear()
location.reload()
```

⚠️ **警告**：此操作会删除所有数据，且不可恢复！

## 使用技巧

### 技巧 1：养成每日记账习惯

- 建议每天固定时间记账（如睡前）
- 及时记录，避免遗忘
- 使用备注功能记录详细信息

### 技巧 2：合理设置预算

- 第一个月先记录，了解消费习惯
- 第二个月再根据实际情况设置预算
- 每季度回顾和调整预算

### 技巧 3：利用备注功能

- 记录详细信息（如："午餐 - 公司食堂"）
- 便于后续分析消费细节
- 可以记录商家名称、地点等

### 技巧 4：定期查看统计

- 每周查看一次分类占比
- 月末回顾全月支出趋势
- 找出可以优化的消费项目

### 技巧 5：添加到主屏幕

将应用添加到手机主屏幕，使用更方便：

**iOS Safari**：
1. 打开应用
2. 点击分享按钮
3. 选择"添加到主屏幕"

**Android Chrome**：
1. 打开应用
2. 点击菜单（三个点）
3. 选择"安装应用"

## 故障排除

### 问题 1：页面无法加载

**解决方案**：
- 检查网络连接（首次加载需要）
- 尝试刷新页面
- 清除浏览器缓存后重试
- 尝试使用其他浏览器

### 问题 2：数据保存失败

**解决方案**：
- 检查浏览器是否启用了 localStorage
- 确认浏览器不是在无痕/隐私模式
- 检查 localStorage 是否已满
- 尝试刷新页面后重试

### 问题 3：图表不显示

**解决方案**：
- 确认已添加至少一笔支出
- 确认支出日期在本月
- 检查浏览器是否支持 Canvas
- 尝试刷新页面

### 问题 4：响应式布局异常

**解决方案**：
- 确认使用推荐的浏览器版本
- 尝试调整浏览器窗口大小
- 在移动设备上尝试横屏/竖屏切换

### 问题 5：编辑按钮无反应

**解决方案**：
- 检查浏览器控制台是否有错误（F12）
- 尝试刷新页面
- 确认支出记录数据格式正确
