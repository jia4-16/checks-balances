# 快速开始指南 🚀

欢迎使用个人记账工具！本指南将帮助您在 5 分钟内上手使用。

---

## 第一步：访问应用

### 在线使用
直接访问我们的 GitHub Pages 地址：
```
https://[username].github.io/checks-balances/
```

### 本地使用
1. 下载项目文件
2. 双击 `index.html` 文件在浏览器中打开
3. 开始使用！

---

## 第二步：设置您的预算

1. 点击页面上的 **"设置预算"** 按钮
2. 在弹出的对话框中输入您的月度预算（例如：5000）
3. 点击 **"保存"**

💡 **建议**：根据您的月收入合理设置预算，一般建议预留 20% 作为应急资金。

---

## 第三步：添加第一笔支出

1. 点击右上角的 **"快速记账"** 按钮
2. 填写支出信息：
   - **金额**：输入支出金额（如 25.50）
   - **类别**：选择合适的类别（如 🍔 餐饮）
   - **日期**：默认为今天，可按需调整
   - **备注**：可选，添加详细说明（如 "午餐 - 麦当劳"）
3. 点击 **"保存"**

🎉 恭喜！您已成功添加第一笔支出记录！

---

## 第四步：查看统计分析

添加几笔支出后，您可以：

### 查看预算使用情况
- 页面顶部会显示：
  - 预算总额
  - 已支出金额
  - 剩余预算
  - 每日可支配额度
  - 进度条和状态提示

### 查看图表
- **分类占比饼图**：了解钱都花在哪里了
- **每日支出趋势图**：查看消费趋势变化

### 筛选记录
- 使用下拉菜单按类别筛选
- 使用时间筛选查看特定时间段的支出

---

## 常用操作速查

### 编辑支出记录
1. 在支出列表中找到要编辑的记录
2. 点击 ✏️ **编辑** 按钮
3. 修改信息后保存

### 删除支出记录
1. 在支出列表中找到要删除的记录
2. 点击 🗑️ **删除** 按钮
3. 在确认对话框中点击 **"删除"**

⚠️ **注意**：删除操作不可恢复，请谨慎操作！
